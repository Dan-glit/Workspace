# ¿Qué es el comit en GT HUB? 
#### De forma similar a guardar un archivo editado, una confirmación registra los cambios en uno o más archivos de tu rama . Git asigna a cada confirmación un ID único, llamado SHA o hash, que identifica: los cambios específicos y cuándo se realizaron.
# ¿Qué es un pull?
#### git pull es un comando de Git que descarga automáticamente los cambios desde un repositorio remoto y los fusiona (hace merge) en tu rama local actual. Es equivalente a ejecutar git fetch seguido de git merge, lo que mantiene tu código de trabajo actualizado con el del servidor. 
# ¿Que es el push?
####  en GitHub es el comando utilizado para transferir (subir) las confirmaciones (commits) o cambios realizados en tu repositorio local (en tu computadora) hacia un repositorio remoto en la nube (como GitHub). Es el paso final para publicar o actualizar tu trabajo, permitiendo que otros colaboren.
